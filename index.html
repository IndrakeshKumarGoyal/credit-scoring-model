Project Overview

How It Works

A complete ML pipeline — from synthetic data generation and feature engineering to model training and interactive risk assessment.

📊

Feature Engineering

Engineered ratios like Debt-to-Income (DTI), Payment-to-Income (PTI), and Savings-to-Income directly from raw financial data to boost model accuracy.

🤖

Classification Models

Trained and compared Logistic Regression, Decision Tree, Random Forest, and Gradient Boosting classifiers using balanced class weights for default prediction.

📈

Comprehensive Evaluation

Assessed model quality with Precision, Recall, F1-Score, and ROC-AUC. Compared performance via ROC and Precision-Recall curves.

🔮

FICO Score Mapping

Converts predicted default probability into a standard credit score (300–850) with risk bands: Exceptional, Very Good, Good, Fair, and Very Poor.

⚡

In-Browser Inference

The trained Logistic Regression model weights are embedded directly in JavaScript — no server required. Works on GitHub Pages instantly.

🛡️

Risk Factor Analysis

Automatically identifies and presents key warning and positive factors influencing the credit decision for each assessed applicant.
